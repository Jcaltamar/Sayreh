You must be on version 1.2 for updating the product to 1.3
Please follow the instructions below to successfully update your product to version 1.3

- Download the fresh copy of Human Resource and Payroll Management System from codecanyon (if you have done it already, skip this step)
- Take a note of the database info that you were using for version 1.3
- Replace all the files and folders of your previous version with the new one downloaded just before except the 'uploads' folder
- Go to the file database.php in application/config/database.php and replace db_host, db_user, db_pass, db_name with your credentials
- Go to the file routes.php in application/config/routes.php and set default controller from 'install' to 'login'
- Go to your web browser and enter your application url and login
- Go to settings > general settings and find the section 'update product'
- Select the zip file 'update_1.3.zip' in 'update_pack/update_1.3.zip' and hit install update
- You will find a field named 'Purchase code' in system settings. Enter your purchase code there and update settings.
